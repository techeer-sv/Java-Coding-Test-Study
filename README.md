# Java-Coding-Test-Study
3기 자바 코딩 테스트 준비 스터디


## Rule
1. 자신의 이름으로 Branch를 생성.
2. 주차별로 폴더를 만들고, 그 안에 해당 주차의 문제 풀이 코드를 Push.
3. 파일 이름은 자신이 푼 문제 이름으로 할 것.   
(예시) wooseok/1주차/problem.java
4. 매주 main branch로 merge시키도록 하겠습니다.
